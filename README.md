# Spectrum Analyzer built with RTL-SDR.

## To run 
```
# Terminal 1 – build/
sudo ./sdr_streamer

# Terminal 2 –  sdr_bridge/
node server.js

# Terminal 3 – ui/sdr/web/
npm run dev
```
